# 😷 Face Mask Detector Web App (CNN)

Aplikasi berbasis web untuk mendeteksi penggunaan masker wajah menggunakan arsitektur Convolutional Neural Network (CNN) dan dideploy menggunakan Streamlit.

## Arsitektur Model
* **Framework:** TensorFlow / Keras
* **Input Size:** 120x120 piksel (RGB)
* **Output:** Binary (Sigmoid activation)

## Cara Menjalankan Secara Lokal
1. Clone repo ini
2. Install requirement: `pip install -r requirements.txt`
3. Jalankan aplikasi: `streamlit run app.py`
